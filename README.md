**Real-Time Data Integration:** Fetches live market prices and fundamental financial statements using the yfinance API.

**Fundamental Health Engine:** Automatically flags stocks with high debt-to-equity ratios (> 1.0) or low profitability (ROE < 15%).

**Portfolio Stress Test:** A "Market Crash Simulator" that calculates potential portfolio drawdown based on a simulated Nifty 50 drop (using Beta-weighted systematic risk).

**Interactive Sidebar:** Seamlessly add and manage holdings with manual inputs for Tickers, Quantity, and Average Buy Price.

**Automated Analytics:** Instant calculation of Total Invested Value, Current Portfolio Value, and Unrealized P&L.

**Tech Stack**
Language: Python 3.9+

Data Processing: Pandas (for data cleaning and vector calculations)

Financial Data: yfinance (Yahoo Finance API)

Frontend/UI: Streamlit (for high-performance data dashboarding)
